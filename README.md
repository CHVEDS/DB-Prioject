# Аналитический центр банковской устойчивости

Приложение для автоматизированного анализа годовых отчетов банков (в формате PDF). Приложение извлекает данные из таблиц баланса и отчета о прибылях и убытках, рассчитывает ключевые банковские коэффициенты и формирует текстовый отчет.

## Особенности

- Извлечение финансовых данных из PDF-файлов годовых отчетов банков
- Расчет ключевых банковских коэффициентов:
  - Достаточность капитала (аналог Н1.0)
  - Ликвидность (мгновенная и текущая)
  - Рентабельность (ROE, ROA, NIM)
  - Качество активов
  - Интегральный индекс устойчивости (BSI)
- Современный веб-интерфейс на Streamlit
- Без использования ИИ-агентов и LLM (вся логика детерминирована)

## Технические требования

- Python 3.11.*
- Библиотеки, указанные в `requirements.txt`

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
streamlit run app.py
```

3. Откройте веб-браузер и перейдите по адресу: http://localhost:8501

## Архитектура

Проект разделен на модули:

- `config.py` — константы, пути, маппинг счетов
- `parser.py` — логика извлечения данных из PDF через MinerU
- `calculator.py` — формулы расчета коэффициентов
- `app.py` — интерфейс Streamlit

## Функциональные возможности

- Загрузка PDF-файлов годовых отчетов банков
- Автоматическое извлечение данных из баланса и отчета о прибылях
- Расчет и визуализация ключевых банковских коэффициентов
- Генерация аналитического отчета
- Возможность экспорта результатов в формате TXT и CSV

## Использованные технологии

- **Streamlit** — веб-интерфейс
- **MinerU** — конвертация PDF в структурированный текст
- **BeautifulSoup4** — парсинг HTML/Markdown
- **Pandas** — обработка табличных данных
- **NumPy** — математические вычисления

## Пример использования

1. Загрузите годовой отчет банка в формате PDF
2. Нажмите кнопку "Анализировать отчет"
3. Ознакомьтесь с извлеченными данными и рассчитанными коэффициентами
4. Скачайте полный аналитический отчет
